# Arpeggiator

An app that uses Tone.js to create an arpeggiator.

**Programming Language**: JavaScript

**Web Framework**: React

**Library**: [Tone.js](https://tonejs.github.io/)

**Markup/Styling**: HTML5/SCSS

An arpeggiator automatically steps through a sequence of notes based on an input chord, thus creating an arpeggio.

The arpeggiator can:
- increase/decrease BPM
- increase/decrease swing
- increase/decrease gain
- Select from a range of synths
- Select from a range of note duration
- Has three note orders - up, down and up & down
- Record and download audio

---

# Setup

- Download the files

- Navigate to project folder and run:

```
npm i
```
```
npm start
```
